# QRDELIVER
Inspired by https://github.com/claudiodangelis/qr-filetransfer.

# Usage
```
$ qrdeliver [filename]
or
$ qrd [filename] //shortcut
```

# Description
Make your file be a QR code that mobile device can open/download it by scanning . 

# Installation
```
$ npm i qrdeliver -g
```
